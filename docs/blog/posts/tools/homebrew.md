# Homebrew

执行以下代码，跟着配置就行

```bash
/bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"
```

---

关闭自动更新

```bash
echo 'export HOMEBREW_NO_AUTO_UPDATE=1' >> ~/.zshrc
```
